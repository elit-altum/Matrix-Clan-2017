Synhronous Prelims

The Task :
Participants have to make an online contact manager, where a user logs in and can immediately access his contacts, add some, or edit the existing.
Details of the contact should compulsorily have a Profile Picture, Name, Contact Info and Occupation. Other aspects can be added if the participant wishes.
The participants should create a relevant logo and name for their Web service.
The website should have a landing page and at least 4 active links including the Home Page linkage.

Judgement:
The website should be focused on user experience as the judgement would have maximum points for the same. Bootstrap would be allowed for the prelims but not for the finals.
Along with UX, judgement will be based upon originality, aesthetic appeal and UI elements.
Usage of templates will lead to immediate disqualification.
Entries have to be submitted by 2359 15th August out of which only 8 teams will qualify for the on site prelims.
Attach a zip file or send a google drive link to thematrixclan@gmail.com for submissions.
